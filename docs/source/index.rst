
LitRevAI documentation
======================

Welcome to the LitRevAI documentation.

LitRevAI is a python package to conduct automated literature reviews,
combining various NLP techniques such as Retrieval Augmented Generation (RAG) and Topic Modelling.

It supports synchronization with Zotero, allowing for easily create collections of literature.

.. note::

   This project is under active development.
